快速开始

1

创建开发者账号

注册开放平台账号并创建应用获取API密钥

$ curl -X POST https://api.openplatform.com/apps
-H "Content-Type: application/json"
-d '{"name": "MyApp", "description": "My Application"}'
2

安装SDK

选择适合您开发语言的SDK并安装

# 使用npm安装JavaScript SDK
npm install openplatform-sdk
3

调用API

使用您的API密钥进行身份验证并调用API

const OpenPlatform = require('openplatform-sdk');
const client = new OpenPlatform({ apiKey: 'YOUR_API_KEY' });

client.getUserData()
.then(data => console.log(data))
.catch(error => console.error(error));
